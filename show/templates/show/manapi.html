API Manual

You can use a GET/POST request to request sentiment Analysis of a text

Make sure you hit the "/api/sentiment" endpoint with the "text" parameter and your text as input.

API Limits: Authenticated Users: 1000/day, Non-authenticated Users: 100/day

Use /api-token-auth endpoint with POST request containing username and password for authentication.
